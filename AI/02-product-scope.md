# Product Scope

## Що входить у першу версію

### Публічна частина
- Головна сторінка
- Про компанію
- Послуги (індекс)
- Окремі сторінки послуг
- Блог / статті (список)
- Окрема сторінка статті
- FAQ
- Контакти
- Сторінка «Ціни / вартість»
- Сторінка «Відгуки»
- Сторінка «Політика конфіденційності»
- Сторінка «Умови використання»
- 404

### Локальні SEO-сторінки
- шаблонні посадкові сторінки під запити типу:
  - адвокат у Львові
  - сімейний адвокат Львів
  - адвокат по спадщині Львів
  - адвокат по розлученню Львів
  - адвокат по аліментах Львів
  - адвокат по цивільних справах Львів
  - адвокат по житлових спорах Львів
  - адвокат по адміністративних справах Львів

### Адмінка
- вхід
- dashboard
- статті
- послуги
- FAQ
- відгуки
- заявки
- SEO-сторінки
- глобальні налаштування
- медіа (мінімально)

## Що не входить у першу версію

- складна ролева модель з багатьма правами доступу
- візуальний page builder
- багатомовність у першій ітерації
- CRM всередині сайту
- онлайн-оплата
- складні особисті кабінети клієнтів
- важкі кастомні анімації на GSAP без чіткої потреби

## MVP-принцип

Усе, що не допомагає:
- SEO,
- швидкості,
- довірі,
- контенту,
- заявкам,

має нижчий пріоритет.
