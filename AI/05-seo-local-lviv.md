# SEO and Local SEO — Lviv

## Головна SEO-ціль

Зробити сайт сильним під локальні комерційні та інформаційні запити по Львову.

## Основні групи сторінок для SEO

### 1. Комерційні сторінки
- адвокат у Львові
- адвокатська контора Львів
- юридична консультація Львів
- представництво в суді Львів

### 2. Практики
- сімейний адвокат Львів
- адвокат по аліментах Львів
- адвокат по розлученню Львів
- спадковий адвокат Львів
- цивільний адвокат Львів
- адвокат по житлових спорах Львів
- адміністративний адвокат Львів

### 3. Інформаційні статті
- як подати на розлучення у Львові
- як оформити спадщину
- що робити при спорі щодо аліментів
- як оскаржити штраф / рішення
- як підготувати позовну заяву

## Принципи SEO-архітектури

1. Для кожної важливої послуги — окрема посадкова сторінка.
2. Для кожного сильного інформаційного наміру — окрема стаття.
3. Статті внутрішньо лінкуються на сторінки послуг.
4. Сторінки послуг лінкуються на релевантні статті.
5. На сторінках має бути чіткий H1, логічна ієрархія H2/H3.
6. Не робити дублікати сторінок під однаковий намір.

## Metadata rules

Для кожної сторінки обов’язково:
- title
- description
- canonical
- open graph image
- robots controls при потребі

## Structured data

### Для сайту в цілому
- Organization
- LegalService або LocalBusiness

### Для статей
- Article
- BlogPosting
- BreadcrumbList

### Для FAQ
- FAQPage

### Для контактів
- LocalBusiness / LegalService з address, contactPoint, areaServed

## Важливі content rules

- тексти повинні бути написані для людей, а не набиті ключами
- пояснення мають бути ясними і юридично акуратними
- FAQ повинен відповідати на реальні питання користувачів
- у статтях варто використовувати приклади сценаріїв, коли це доречно
- абзаци короткі, легко скануються

## Internal linking rules

- у кожній статті мінімум 2–4 внутрішні посилання
- у кожної послуги блок «Корисні статті»
- у локальних SEO сторінок блок «Пов’язані послуги»

## Slug rules

- тільки lower-case
- через дефіс
- без транслітераційного хаосу
- коротко, але зрозуміло

## Приклади slug
- `simeynyy-advokat-lviv`
- `spadkovi-spravy-lviv`
- `yak-podaty-na-rozluchennya-u-lvovi`

## Контентний мінімум на старт

### Сторінки послуг
8–12 сторінок

### Статті
15–25 стартових статей

### FAQ
20–40 питань, розбитих по категоріях
