# AI Docs Index — Klimenko & CO

Це центральна точка входу для AI-агента та людей, які працюють над сайтом **Адвокатська контора Клименко & CO**.

## Мета проєкту

Побудувати швидкий, SEO-оптимізований сайт юридичної компанії під **місто Львів** на стеку:

- Next.js (App Router)
- PostgreSQL
- Prisma
- Auth.js
- власна невелика admin panel на Next.js
- markdown / MDX або block editor тільки для статей
- весь інший контент — через структуровані поля в БД

## Пріоритети

1. Висока швидкість завантаження
2. Сильне локальне SEO під Львів
3. Зручне редагування контенту через свою адмінку
4. Чистий, не-штучний дизайн
5. Простий масштабований код
6. Повна передбачуваність для AI-агента
